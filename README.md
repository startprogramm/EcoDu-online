# EcoDu - Environmental Education Platform

An interactive educational platform for learning about ecology and environmental issues.

## Features

- User Authentication (Login/Register)
- User Profiles with Progress Tracking
- Video Lessons on Environmental Topics
- Interactive Quizzes with Results Tracking
- Comments System on Lessons
- Responsive Design

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file with your Supabase credentials:
```env
VITE_SUPABASE_URL=your_url_here
VITE_SUPABASE_ANON_KEY=your_key_here
```

3. Run development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
